HOW TO SETUP (DO THIS ONLY ONCE ON ONE COMPUTER):
──────────────────────────────────────────────────

STEP 1: Install Node.js
  - Download from: https://nodejs.org
  - Download the LTS (Long Term Support) version
  - Install it (just click Next → Next → Install)

STEP 2: Extract this ZIP to a permanent folder
  - Example: C:\TMH_Server\
  - Keep this folder safe — all uploaded data is stored here

STEP 3: Double-click "START_SERVER.bat"
  - A black window will appear showing the server IP addresses
  - Note down the IP shown (e.g., 10.9.2.53)
  - KEEP THIS WINDOW OPEN while using the app on other PCs

STEP 4: On all other PCs (including this one)
  - Open the HTML file (TMH_PDF_Tool.html) in browser
  - Enter the IP address shown in the server window
  - Click "Test" to connect
  - Now all PCs share the same data!

═══════════════════════════════════════════════════════════
IMPORTANT NOTES:
  ✅ Server PC must be ON for other PCs to access data
  ✅ All PDFs are stored in the "tmh_data" folder
  ✅ Do NOT delete the tmh_data folder
  ✅ To start server daily: double-click START_SERVER.bat
